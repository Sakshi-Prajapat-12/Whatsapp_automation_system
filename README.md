
# whatsapp Automation System
This project is a **WhatsApp Automation System** built using Python and Flask.  
It allows users to send  WhatsApp messages and images using contact data from Excel/CSV files.

## Tech stack 
python 3.12
Mysql
Pandas
Pywhatkit
Flask
Html
css
